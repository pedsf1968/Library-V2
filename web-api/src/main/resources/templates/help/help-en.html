Information system project for the library of a big city.

This RELEASE consists of the WEB application for users and a batch which performs reminders.

Any user can consult the different media present in the library.

There are four different types of media: books, games, music and video

A user can register to become a member and borrow media

A member can view their loans

A member can reserve a media in order to be able to remove and borrow it

A member can extend his loan when it expires

A reminder email is sent for each delay.
